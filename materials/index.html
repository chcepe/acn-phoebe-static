AI Overview

Artificial Intelligence is one of the emerging technologies of the 4th industrial revolution. It is all around us from self-driving cars and drones to virtual assistants and software that translate or invest. Driven by exponential increases in computing power and by the availability of vast amounts of data, Impressive progress has been made in AI in the recent years. AI is a very broad area which consists of multiple technologies that enables machines to sense, comprehend, act and learn.

It can answer questions from the users, provides basic troubleshooting, and can give suggestions or recommendations. It uses the concept of Natural Language Processing to understand and to recognize human languages.

Computer Vision

Computer Vision is a science that is concerned with the theory and technology of computers having a high-level understanding of the information from digital images or videos. It aims to automate tasks that a human visual system can do. These tasks include acquiring, analyzing and processing high-dimensional data from the real world.

Object Detection

Object detection is a technology related to image processing that finds instances of real-world objects such as tables, chairs, humans, cars, etc.) Tensorflow Object Detection API is an open-source framework built on top of Tensorflow that makes it easy to construct, train and deploy models for different purposes. YOLO (You Only Look Once) is one of the networks for object detection. It is faster than the earlier methods such as RCNN and fast RCNN

MEAN Stack

MEAN Stack (MongoDB, ExpressJS, AngularJS, NodeJS) is an open-source JavaScript software stack for building web sites and web applications. It allows development teams to create projects utilizing JavaScript across all tools. MongoDB is a NoSQL document datastore that uses JSON-style documents. Express is an HTTP server framework on Node.js that is used for Web application development. Angular is a framework for frontend JavaScript.
